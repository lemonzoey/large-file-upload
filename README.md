# 介绍
大文件上传教程
# 技术架构

vue3 + vite + axios  

搭配node.js + express 使用

# 使用说明

1.文件拉取下来后，npm install 安装前后端所需依赖

2.前端项目 v3-file-upload, 后端项目:node-server

3.前端项目的启动命令是 npm run dev

4.node启动命令是：npm run dev

5.node >= 14
